 10: /**
 11:  * It defines some handful methods and enhances object core of PHP:
 12:  *   - access to undeclared members throws exceptions
 13:  *   - support for conventional properties with getters and setters
 14:  *
 15:  * Properties is a syntactic sugar which allows access public getter and setter
 16:  * methods as normal object variables. A property is defined by a getter method
 17:  * or setter method (no setter method means read-only property).
 18:  * <code>
 19:  * $val = $obj->label; // equivalent to $val = $obj->getLabel();
 20:  * $obj->label = 'Nette'; // equivalent to $obj->setLabel('Nette');
 21:  * </code>
 22:  * Property names are case-sensitive, and they are written in the camelCaps
 23:  * or PascalCaps.
 24:  *
 25:  * @author David Grudl
 26:  * @copyright Nette Framework
 27:  * @author Petr Procházka
 28:  * @package Orm
