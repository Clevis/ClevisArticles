 10: /**
 11:  * Handles entities.
 12:  * Independently of the specific storage.
 13:  * Saving, deleting, loading entities.
 14:  *
 15:  * For each entity type (or group of related entities) you must create own repository.
 16:  *
 17:  * Convention is named repository plural form and entity singular form.
 18:  *
 19:  * <code>
 20:  * class ArticlesRepository extends Repository
 21:  * </code>
 22:  *
 23:  * Repository must be obtained via IRepositoryContainer {@see RepositoryContainer}
 24:  * <code>
 25:  * $model; // instanceof RepositoryContainer
 26:  * $model->articles; // instanceof ArticlesRepository
 27:  * </code>
 28:  *
 29:  * Repository is independently of the specific storage.
 30:  * About storage is cares Mapper {@see IMapper} {@see DibiMapper}
 31:  *
 32:  * Naming convention methods for retrieving data:
 33:  * `getBy<...>()` for one entity {@see IEntity}
 34:  * `findBy<...>()` for collection of entities {@see IEntityCollection}
 35:  * `findAll()` all entities
 36:  *
 37:  * You can automatically call methods in mapper like `$mapper->findByAuthorAndTag($author, $tag)` etc.
 38:  * But in repository is needed to create all methods:
 39:  * <code>
 40:  * public function findByAuthor($author)
 41:  * {
 42:  *  return $this->mapper->findByAuthor($author);
 43:  * }
 44:  * public function getByName($name)
 45:  * {
 46:  *  return $this->mapper->getByName($name);
 47:  * }
 48:  * </code>
 49:  *
 50:  * Defaults repository works with entity named by repository name in singular form `ArticlesRepository > Article` {@see self::getEntityClassName()}.
 51:  *
 52:  * Defaults tries find mapper by repository name `ArticlesRepository > ArticlesMapper`
 53:  * It can be changed by annotation `@mappper`.
 54:  *
 55:  * @see self::getById() Get one entity by primary key.
 56:  * @see self::persist() Saving.
 57:  * @see self::remove() Deleting.
 58:  * @see self::flush() Make changes in storage.
 59:  * @see self::clean() Clear changes in storage.
 60:  * @author Petr Procházka
 61:  * @package Orm
 62:  * @subpackage Repository
 63:  */
