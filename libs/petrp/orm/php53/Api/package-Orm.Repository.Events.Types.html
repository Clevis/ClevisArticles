Package Orm

Subpackage Repository\Events\Types

Interfaces summary
Orm\IListener Abstract listener. Use any of subinterface.
Orm\IListenerAttach Entity is attached to repository. Has EventArguments::$entity.
Orm\IListenerCleanAfter It fires after repository was cleaned.
Orm\IListenerCleanBefore It fires before repository will be cleaned.
Orm\IListenerFlushAfter It fires after repository was flushed.
Orm\IListenerFlushBefore It fires before repository will be flushed.
Orm\IListenerHydrateAfter After data from storage are hydrated into entity. Has EventArguments::$data and EventArguments::$entity. It fires after data are hydrated. Entity has all values. If $data are changed there's no effect.
Orm\IListenerHydrateBefore Before data from storage are hydrated into entity. Has EventArguments::$data and EventArguments::$entity. It fires before data are hydrated. Entity is empty. $data can be changed. Values at entity will be overwritten.
Orm\IListenerPersist Entity is saved. Has EventArguments::$id and EventArguments::$entity. $id can be changed but it's not recommended. Some relationship are not persisted yet. If entity is changed during Events::PERSIST_AFTER, this event will fired twice.
Orm\IListenerPersistAfter Entity was saved. It fires after. Has EventArguments::$entity. If entity is changed during this event, changes will be updated in storage and it fires Events::PERSIST again.
Orm\IListenerPersistAfterInsert Entity was saved first time. It fires after. Has EventArguments::$entity. If entity is changed during this event, changes will be updated in storage and it fires Events::PERSIST again.
Orm\IListenerPersistAfterUpdate Entity was updated. It fires after. Has EventArguments::$entity. If entity is changed during this event, changes will be updated in storage and it fires Events::PERSIST again.
Orm\IListenerPersistBefore Entity is changed and will be saved. It fires before. Has EventArguments::$entity.
Orm\IListenerPersistBeforeInsert Entity will be saved first time. It fires before. Has EventArguments::$entity.
Orm\IListenerPersistBeforeUpdate Entity will be updated. It fires before. Has EventArguments::$entity.
Orm\IListenerRemoveAfter It fires after entity was deleted. Has EventArguments::$entity.
Orm\IListenerRemoveBefore It fires before entity will be deleted. Has EventArguments::$entity.
Orm\IListenerSerializeAfter After entity is serialized to storage. Has EventArguments::$values, EventArguments::$operation and EventArguments::$entity. EventArguments::$values contains all scalarized values. EventArguments::$operation contains string insert or update depending on entity is (or not) persisted first time.
Orm\IListenerSerializeBefore Before entity is serialized to storage. Has EventArguments::$params, EventArguments::$values, EventArguments::$operation and EventArguments::$entity. EventArguments::$params are only in DibiMapper and contains DibiPersistenceHelper::$params. EventArguments::$values contains all entity values given by IEntity::toArray() before then will scalarized. EventArguments::$operation contains string insert or update depending on entity is (or not) persisted first time.
Orm\IListenerSerializeConventional After entity is serialized and conventional is applied. Has EventArguments::$values, EventArguments::$operation and EventArguments::$entity. EventArguments::$values contains all scalarized values. Conventional is applied. EventArguments::$operation contains string insert or update depending on entity is (or not) persisted first time.
